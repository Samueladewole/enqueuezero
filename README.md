# EnqueueZero

Enqueue Zero is creating fun arts on Computer Science and Programming. It's operated by a single man team since 2018.

## Development

Create environment

```
# Python 3.7 required
$ brew install python

$ pip install pipenv

$ pipenv install

$ pipenv shell

(env) $ make devserver
(env) $ ./develop_server.sh stop
(env) $ make github
```
